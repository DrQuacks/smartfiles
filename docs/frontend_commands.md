# SmartFiles Frontend Commands

This document collects common commands for working on the React/Vite frontend.

## Environment

```bash
cd /Users/Kellar/Develop/Projects/smartfiles/frontend
```

Install dependencies:

```bash
npm install
```

## Development

Run the dev server (Vite):

```bash
npm run dev
```

Build the frontend for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Lint the frontend codebase:

```bash
npm run lint
```
